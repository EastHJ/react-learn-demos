React 的 JSX 语语法允许HTML元素与JavaScript代码混写而不用将HTML元素转换成字符串
JSX语法的转移字符基本规则是：
    遇到 HTML 标签（以 < 开头），就用 HTML 规则解析；
    遇到代码块（以 { 开头），就用 JavaScript 规则解析。

代码

var names = ['Alice','Lily','Jack'];
var element = (
    <div>
        {
            names.map(
                function (name) {
                return <div>Hello,{name}</div>
                }
             )
         }
    </div>
);
ReactDOM.render(element,document.getElementById('root'));
